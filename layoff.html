Layoff Data Analysis (SQL)

View on GitHub

Business Problem

Understand patterns and trends in global layoffs across industries and regions.

Approach

Data cleaning, aggregation, window functions and reporting using SQL.

Key Insights

  • Tech sector had highest layoffs
  • Startups in later stages saw more layoffs
  • US accounted for majority of records

Tools Used

MySQL, SQL (CTE, window functions), Excel
