Power BI Case Study – Data Professional Survey Dashboard

Survey | Data Analysis | Visualization | Power BI

1. Project Overview

This Power BI dashboard provides insights into the demographics, salary trends, career satisfaction, and industry distribution of over 600 data professionals. The objective was to transform raw survey results into an interactive decision-support dashboard for HR teams and aspiring data professionals.

2. Dataset Description

The dataset includes responses on job roles, countries, difficulty breaking into data roles, salaries, work–life balance ratings, happiness scores, programming preferences, and industry sectors.

3. Tools Used

  • Power BI Desktop
  • DAX for calculated measures
  • Power Query for data cleaning
  • Excel (source data)

4. Analysis Steps

  • Cleaned and standardized survey data in Power Query
  • Created custom measures (Avg Salary, Satisfaction Scores)
  • Built interactive visuals including treemaps, donut charts, KPIs, line charts, and bar charts
  • Added slicers for country, job title, and industry filtering

6. Insights

  • Average salary of Data Scientists is significantly higher than other roles.
  • Majority of survey takers are from the United States, India, and the UK.
  • Most professionals find it “neither easy nor difficult” to break into data roles.
  • Work–life balance satisfaction is moderately high across roles.
  • Programming preferences vary significantly by job role.

7. Conclusion

The dashboard offers organizations a quick view into hiring trends, skill preferences, job satisfaction, and salaries. This project enhanced my ability to design professional dashboards with DAX-driven insights.
