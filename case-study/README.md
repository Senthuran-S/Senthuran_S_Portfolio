# Layoffs Data Cleaning — Case Study (SQL)

This case study documents the full end-to-end workflow of cleaning the global **Layoffs Dataset**, which contains thousands of records from companies across multiple industries and countries.

## Objectives
- Remove duplicate and inconsistent entries  
- Standardize company, industry, and country fields  
- Fix incorrect date formats  
- Handle missing/null values  
- Prepare dataset for analysis and dashboarding
