Excel Case Study – Bike Sales Dashboard

Excel | Pivot Tables | Dashboarding | Data Cleaning

1. Project Overview

This Excel dashboard analyzes customer patterns related to bike purchases. The goal was to uncover which demographic groups show higher purchasing likelihood and create a clean, interactive dashboard using Excel-only tools.

2. Dataset Description

The dataset contains demographic attributes such as gender, marital status, occupation, region, and commute distance, along with whether each customer purchased a bike.

3. Tools Used

  • Excel Pivot Tables
  • Pivot Charts
  • Data Validation
  • Interactive slicers

4. Analysis Steps

  • Performed data cleaning and standardized categorical values
  • Built PivotTables for income, commute distance, and age-group segmentation
  • Developed interactive slicers for region, marital status, and occupation
  • Designed a dynamic dashboard layout

6. Key Insights

  • Married customers have a higher bike purchasing rate compared to single customers.
  • Commute distance significantly impacts purchasing decisions — shorter commutes correlate with more purchases.
  • Females with higher income levels are more likely to purchase a bike.
  • Skilled manual and clerical occupations show stronger engagement in purchasing.

7. Conclusion

The project demonstrates the ability to build meaningful dashboards in Excel without advanced BI tools. By leveraging pivot tables and slicers, complex insights were transformed into an intuitive dashboard.
